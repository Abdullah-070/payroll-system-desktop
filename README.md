# Employee Payroll System - Desktop Version

A complete payroll management solution with role-based access control.

## Features

**Admin Access**
- Manage employees (add, edit, delete, search)
- Process payroll and calculate salaries
- Generate payroll reports and statistics
- Print employee pay slips
- Sort and view employee data

**Employee Access**
- View personal profile
- Check salary information
- View salary history and pay slips

## Technology Stack
- Language: Python 3.7+
- Database: SQLite3
- GUI: tkinter + tkcalendar
- Security: SHA-256 password hashing

## Default Credentials
- Admin: `admin` / `admin123`
- Employee: Create account via admin dashboard

## Quick Start
```bash
python payroll_gui.py
```
